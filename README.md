# tuck-site

The marketing and docs site for [tuck](https://github.com/Pranav-Karra-3301/tuck), the modern dotfiles manager. tuck is a fast, TypeScript-based CLI that tracks, syncs, and shares any config file (in your home directory or inside any git repo) across machines using Git. It is agent-native too, with structured JSON output on every command and a write-confined sandbox.

Live site: [tuck.sh](https://tuck.sh)

## Tech

Built with [Next.js](https://nextjs.org) (App Router) and TypeScript.

## Development

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

Other scripts:

```bash
npm run build   # production build
npm run start   # serve the production build
npm run lint    # eslint
```
